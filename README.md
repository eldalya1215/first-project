# AI 기반 50대 여성 맞춤 운동 상담 및 운동 계획 자동화

## 프로젝트 개요

본 프로젝트는 생성형 AI를 활용하여 **50대 여성 맞춤 운동 상담과 하루 5분 운동 루틴을 자동 생성**하는 업무를 수행하기 위해 진행하였다.

동일한 프롬프트를 **GPT-5.5, Gemini 2.5 Pro, Claude 4 Sonnet**에 입력하여 결과를 비교하고 실제 업무에 가장 적합한 모델을 선정하였다.

---

# 1. 업무 정의

### 업무명

AI 기반 50대 여성 맞춤 운동 상담 및 운동 계획 자동화

### 대상 사용자

- 50대 여성
- 허리 통증이 있는 운동 초보자
- 홈트레이닝 사용자
- 운동 지도자
- 건강 콘텐츠 제작자

### 업무 목적

AI를 활용하여 상담 시간을 줄이고 운동 프로그램을 자동 생성하여 실제 회원 상담 및 콘텐츠 제작 업무의 효율성을 높인다.

---

# 2. 입력(Input)

사용자가 입력하는 정보

- 연령
- 성별
- 통증 부위
- 운동 경험
- 운동 목적

### 입력 예시

```
50대 여성을 위한 하루 5분 허리 운동 루틴을 만들어 주세요.

운동 초보자도 쉽게 따라 할 수 있도록 설명하고,

주의사항과 응원 메시지도 함께 작성해 주세요.
```

---

AI가 생성하는 결과

- 운동 루틴
- 운동 순서
- 운동 설명
- 주의사항
- 응원 메시지
- 유튜브 쇼츠 대본
- 콘텐츠 제작 스크립트
