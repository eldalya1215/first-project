회의 메모

우리가 만들 앱은 50대 여성들이 혼자 운동을 꾸준히 할 수 있도록 도와주는 AI 운동 코칭 앱이다.
회원들은 운동 방법보다 매일 동기부여가 더 어렵다고 이야기한다.
앱은 개인정보 입력을 최소화하고 가입도 간단해야 한다.
첫 화면에서는 오늘 해야 할 운동 하나만 보여주는 것이 좋겠다.
운동시간은 5분부터 시작하도록 한다.
목표를 너무 크게 잡으면 포기하기 때문에 작은 성공 경험을 만들어주는 것이 중요하다.
AI가 오늘 컨디션을 질문하고 운동을 추천하면 좋겠다.
사용자가 운동을 완료하면 칭찬 메시지와 작은 배지를 제공한다.
일주일 동안 운동하면 새로운 챌린지가 열린다.
운동은 무릎이 약한 사람도 가능한 동작을 우선 제공한다.
향후에는 식단관리 기능도 추가하지만 이번 버전에서는 제외한다.
회원들의 후기를 쉽게 남길 수 있도록 한다.
알림은 하루 한 번만 보내고 사용자가 시간을 직접 선택할 수 있게 한다.
초기 개발 목표는 안드로이드 버전이며 3개월 안에 MVP를 완성한다.
마케팅은 유튜브 '몸이 먼저 웃는다' 채널과 티스토리 블로그를 활용한다.
