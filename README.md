## ② 시스템 설계 문서 및 모델 비교 보고서 (Word 보완용 핵심 요약)

위 README 내용을 바탕으로 Word 서류에 추가 및 수정 배치해야 하는 명확한 구조적 가이드라인입니다.

### 1. 프롬프트 다이어그램 구조


* **업무 정의**: 시니어 맞춤형 헬스케어 자동화 챗봇 인터페이스 구축.
* **시스템 프롬프트 (v2)**: 단순 지시문에서 탈피하여 **역할 명시 + 제약 조건(Constraints) + 안전 필터**의 3단 구조로 고도화.
* **Few-shot (3개 고정)**: 
  1. 관절염 환자 ➡️ 체중 부하 운동 제외 처방 예시

  3. 근감소증 환자 ➡️ 점진적 과부하 저항성 운동 처방 예시
