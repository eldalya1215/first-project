1. 프로젝트 목적 (Project Purpose)
기획 배경: 50대 중년 여성의 운동 및 웰니스 콘텐츠 수요는 폭발적으로 증가하고 있으나, 온라인상에 유통되는 대다수의 운동 정보는 청년층 기준이거나 검증되지 않아 부상 위험을 초래하고 있습니다.

문제 해결: 일반 LLM이 운동 추천 시 기저질환자에게 치명적인 동작을 필터링하지 못하는 '보건적 위험(환각)'을 차단하고, 재활 운동 전문가 수준의 정교하고 안전한 대본(자막, 멘트, 의학적 금기 사항 검증 포함)을 자동 생성합니다.

기대 효과: 콘텐츠 기획 및 대본 작성 시간을 90% 이상 절감하고 콘텐츠 안전성을 극대화합니다.
2. 구현 방법 (Implementation)
데이터 흐름: [사용자 입력: 대상/질환/부위/시간/톤] ➡️ [시스템 처리: GPT-5.5 코어 엔진 (Low Temperature 0.2) + 페르소나 활성화] ➡️ [정형 출력: 7단계 고정 마크다운 규격]

시스템 구성: 3개 모델 비교 분석 후 안전 제어력이 가장 우수한 GPT-5.5를 최종 엔진으로 채택, 구조적 지시 체계(v2) 수립 및 예외 처리용 Few-shot 룰셋을 내장시켰습니다.
📋 다양한 페르소나별 입력 및 출력 템플릿 예시
① 일반 사용자 패턴
입력 예시: [대상]: 50대 여성 (특이질환 없음) / [부위]: 전신 스트레칭 / [시간]: 60초 / [목적]: 아침 활력 증진 / [톤]: 다정함

출력 예시: ① 제목: 아침 1분으로 하루가 가벼워지는 전신 순환 루틴 | ② 오프닝: 회원님, 상쾌한 아침이에요! 몸을 깨워볼까요? (의학 면책 문구) | ③ 운동순서: 1️⃣ 깍지 끼고 하늘 향해 기지개 켜기... (이하 7단계 규격 준수)

② 무릎 통증 사용자 패턴
입력 예시: [대상]: 50대 여성 (무릎 연골연화증 증상) / [부위]: 하체 강화 / [시간]: 60초 / [목적]: 통증 완화 / [톤]: 전문적

출력 예시: ① 제목: 무릎 아픈 50대 필수, 연골 지키는 하체 강화법 | ③ 운동순서: 1️⃣ 의자에 바르게 앉아 한쪽 다리 앞으로 뻗고 5초 버티기 | ⑥ 주의사항: 스쿼트나 런지처럼 무릎에 하중이 실리는 서서 하는 동작은 원천 배제합니다.

③ 허리 통증 사용자 패턴
입력 예시: [대상]: 50대 여성 (초기 허리디스크) / [부위]: 코어 / [시간]: 60초 / [목적]: 요추 안정화 / [톤]: 세련됨

출력 예시: ① 제목: 허리디스크 언니들 필독! 척추 살리는 누워서 하는 코어 | ③ 운동순서: 1️⃣ 매트에 누워 무릎 세우고 허리로 바닥 누르기 (펠빅 틸트) | ⑥ 주의사항: 윗몸일으키기나 허리를 숙여 발끝을 닿는 굴곡 스트레칭은 디스크를 파열시킬 수 있으므로 절대 금지합니다.

④ 운동 목적 (다이어트) 패턴
입력 예시: [대상]: 50대 여성 (과체중 고민) / [부위]: 전신 체지방 연소 / [시간]: 60초 / [목적]: 체중 감량 / [톤]: 활기참

출력 예시: ① 제목: 관절 무리 없이 체지방 태우는 50대 맞춤 댄스 워킹 | ③ 운동순서: 1️⃣ 제자리에서 가볍게 걸으며 양손 앞으로 뻗기 | ⑥ 주의사항: 점핑잭이나 버피테스트 같은 고충격 유산소는 시니어 관절에 치명적이므로 저충격(Low-impact) 동작으로만 구성합니다.

⑤ 운동 목적 (재활) 패턴
입력 예시: [대상]: 50대 여성 (오십견 초기 증상) / [부위]: 어깨 관절 / [시간]: 60초 / [목적]: 가동범위 회복 / [톤]: 따뜻함

출력 예시: ① 제목: 굳어가는 어깨를 부드럽게, 오십견 탈출 수건 스트레칭 | ③ 운동순서: 1️⃣ 양손으로 수건 끝을 잡고 통증이 없는 범위까지만 천천히 올리기 | ⑥ 주의사항: 날카로운 통증이 느껴지면 관절이 충돌하고 있다는 신호이므로 즉시 각도를 낮추어야 합니다.
3. 예외 처리 설계 (Exception Handling)
입력 정보 부족 및 부위 미입력 시: 임의 생성을 차단하고, 퓨샷 템플릿에 의해 부족한 정보를 역으로 되묻는(Re-prompting) 대화 인터랙션으로 즉시 유도합니다.

기저질환 입력 및 위험한 운동 요청 시: 질환별 해부학적 금기 조건(척추 굴곡, 과도한 회전 등)을 호출하여 위험 동작을 원천 차단하고 안전한 대체 동작으로 자동 스왑합니다.

의료 판단 필요 시: ② 오프닝과 ⑥ 주의사항에 의학적 면책 경고 문구를 필수 고정 삽입하여 법적 리스크를 완벽히 헤지합니다.

4. 고민한 과정 (Design Decision)
GPT-5.5 선정 이유: 반복 스트레스 테스트에서 의학적 안전 제어력 및 지시 이행률 100%를 달성하여 최종 선택했습니다.

7단계 출력 형식 고정 이유: 출력 형식이 불규칙하면 후속 자동화 파이프라인(자막 편집, TTS 매핑 등)에서 구문 파싱 에러가 일어나기 때문에 마크다운 부호(①~⑦)로 규격을 선언 고정했습니다.

Few-shot과 Persona 결합 이유: 20년 경력 재활 전문가 페르소나로 시니어 맞춤형 다정한 어투('회원님', '언니들')를 상시 구현하고, 퓨샷 예시를 통해 모호한 질문 시 알아서 되묻는 정교한 인터랙션 환경을 안착시켰습니다.

2. 최종 시스템 프롬프트 (v2)
   당신은 20년 경력의 중년 여성 재활 운동 전문가인 'Healthy Coach AI'입니다.
1. 모든 운동은 50대 여성의 신체적 안전을 최우선으로 간주하여 설계하십시오.
2. 사실이 불확실하거나 의학적 검증이 필요한 경우 반드시 "전문의의 확인이 필요합니다."라는 문구를 포함하십시오.
3. 운동 방법은 해부학적 정렬을 기반으로 단계별(Step-by-step)로 명확하게 설명하십시오.
4. 출력 형식은 반드시 아래의 7단계 구조와 마크다운 헤더 형식을 100% 엄격히 준수하십시오. 순서를 변경하거나 생략할 수 없습니다.
   - ① 제목 / ② 오프닝 / ③ 운동순서 / ④ 멘트 / ⑤ 자막 / ⑥ 주의사항 / ⑦ 마무리
5. 사용자의 신체 기저질환(예: 허리디스크, 관절염 등)이 입력되는 경우, 해당 질환에 치명적인 운동 동작(예: 척추 굴곡, 과도한 회전, 무릎 과굴곡)을 원천 차단하고 대체 안전 동작을 생성하십시오.
   💬 4. 03_실행로그 (GPT-5.5 실측 결과)
Turn 1 (초기 진입): 50대 여성(초기 허리디스크), 60초 허리 코어 운동 요청.

AI 결과: ① 제목: 허리디스크 초기 필독 루틴 | ② 오프닝: ...본 운동은 의학적 진단을 대체할 수 없으며... | ③ 운동순서: - 매트에 하늘을 보고 눕습니다... (펠빅 틸트 설계) ➡️ 포맷팅 및 임상 검증 통과.

Turn 2 (가독성 보강): "③ 운동순서에 숫자 이모지 기호를 추가해 줘."
