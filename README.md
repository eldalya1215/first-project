📑 50대 여성을 위한 건강운동 콘텐츠 자동 생성 시스템 구축
본 프로젝트는 스포츠 의학적 전문 지식과 LLM(대형 언어 모델) 프롬프트 엔지니어링 기술을 결합하여, 50대 중장년층 여성에게 최적화된 안전하고 효과적인 건강운동 숏폼 콘텐츠를 자동 생성하는 시스템입니다.

평가자가 3~5분 이내에 프로젝트의 핵심 가치, 기술적 구현 체계, 예외 처리 설계 및 아키텍처를 직관적으로 파악할 수 있도록 구조화한 최종 제출용 마스터 가이드입니다.

1. 프로젝트 목적 (Project Purpose)
📌 기획 배경 및 해결하고자 하는 문제
시니어 시장의 급성장과 정보의 비대칭성: 50대 중년 여성의 운동 및 웰니스 콘텐츠 수요는 폭발적으로 증가하고 있으나, 온라인상에 유통되는 대다수의 운동 정보는 청년층 기준이거나 검증되지 않아 부상 위험을 초래하고 있습니다.

전문 콘텐츠 제작의 리소스 한계: 재활 운동 전문가 수준의 정교하고 안전한 대본(자막, 멘트, 의학적 금기 사항 검증 포함)을 매번 수작업으로 기획·집필하는 데는 막대한 시간과 비용이 소모됩니다.

LLM의 환각(Hallucination) 위험: 일반적인 AI에게 운동 추천을 요청할 경우, 허리디스크나 무릎 관절염 환자에게 치명적인 동작(예: 요추를 과하게 숙이는 굴곡 스트레칭, 스쿼트 등)을 필터링하지 못하고 그대로 추천하는 기술적 위험성이 존재합니다.

💡 해결 방안 및 기대 효과
스포츠 재활 전문가 페르소나와 엄격한 제약 프롬프트, 규칙 기반 시스템을 적용하여 안전성이 100% 검증된 개인 맞춤형 운동 대본 생성 프로세스를 자동화합니다.

숏폼 플랫폼 규격에 최적화된 고부가가치 콘텐츠를 단 몇 초 만에 생산함으로써 콘텐츠 기획 및 대본 작성 시간을 90% 이상 절감합니다.

2. 구현 방법 (Implementation)
본 시스템은 다차원 프롬프트 엔지니어링 기법을 적용하여 입력 데이터가 최종 마크다운 가이드로 변환되기까지 유기적이고 안전하게 동작하도록 구축되었습니다.

스펙 정의안전 필터 레이어"위험한 동작은 삼가해 줘" 수준의 권고특정 질환 매칭 시 해당 해부학적 역동작 차단 하드코딩 수칙이상 입력 방어빈 입력이나 오류 정보 기입 시 에러 및 환각 생성퓨샷(Few-shot) 기반 역질문 룰셋 내장💡 변경 이유 및 성능 향상 결과변경 이유: 초기 v1 버전에서는 사용자가 운동 정보를 누락하거나 잘못된 위험 동작을 요구할 경우, LLM이 이를 거부하지 못하고 그럴듯한 거짓말로 위험한 운동 루틴을 생성하는 심각한 환각 현상이 발생했습니다. 또한, 출력 포맷이 매번 달라져 파이프라인 자동화가 불가능했습니다.성능 향상 및 적용 결과: 시스템 프롬프트를 v2로 고도화한 후, 정형 포맷 이행률이 100%로 상승하여 후속 자막 편집 툴과의 API 연동 에러가 제로로 수렴했습니다. 특히 "허리디스크 환자용 윗몸일으키기"와 같은 악성 입력 시, AI가 즉시 시스템 거부권을 발동하고 안전 대안 동작인 '펠빅 틸트'로 자동 스왑해 주는 임상 안전성 버퍼를 완벽하게 확보했습니다.6. 프로젝트 구조저장소는 아래와 같이 직관적이고 표준화된 구조로 구성되어 있으며, 문서 및 산출물 파일은 일관성 있게 정돈되어 있습니다.
